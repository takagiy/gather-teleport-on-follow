# Privacy Policy — Gather Town Teleport on Follow

**Extension:** Gather Town Teleport on Follow
**Last updated:** 2026-05-18

## Summary

Gather Town Teleport on Follow does not collect, store, transmit, sell, or share any user data.

## Data we collect

**None.** The extension does not collect any personal information, browsing history, authentication credentials, location data, financial information, communications, or any other form of user data.

## Data we transmit

The extension only invokes Gather Town's client APIs for moving the user's character, and does not send data to any server other than Gather's own.

## Permissions used

The extension declares no Chrome `permissions` and no `host_permissions`. It only registers a content script that runs on `https://app.gather.town/*` in the page's MAIN world, which interacts with the Gather Town client to teleport the user when they choose to follow another player.

## Third parties

The extension does not use any third-party analytics, error reporting, advertising, or tracking services.

## Remote code

The extension does not load or execute any remote code. All code is bundled at build time and shipped inside the extension package.

## Changes to this policy

If material changes are made, this file will be updated in place and the **Last updated** date above will be revised. The current version of this policy is always available at the URL provided in the Chrome Web Store Developer Dashboard.
